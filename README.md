### 기능구현
1. 리액트 컴포넌트 분리
2. '오늘의 카카오 날짜 소식' Date로 구현
3. 다크모드 구현
4. 스크롤, 클릭, 호버, 검색 모달 등 애니메이션 구현
5. 반응형, 웹표준성 준수
6. 더미데이터 json 만들어 반복렌더링으로 카드 배너, nav, footer 구현
